# Research Workflow

## Tools
Tools to be used in this workflow.
* To convert a file .md to .pdf use the tool at "../text2podcast"
* To convert a file .md to .mp3 use the tool at "../text2podcast"
* To send a file use the tool at "../text2podcast"

## Research rules
* If giving a number of a scale, provide also note of the percentage of the full scale
* Every research file shall contain a short summary of its focus at the beginning

## Making a Research
* When asked to research a topic do the following steps from 1 till the last.

1. Research the topic in depth on the internet.
2. For every research, create a new directory with the research name
3. Inside the research directory create a research_name.md and put the research in it.
4. Create research_name_podcast.md from the research_name.md, which is basically the same, but charts, tables and so are read so it is understandable to somebody who just listens to it and doen'st see it.
5. Convert the research_name.md to research_name.pdf
6. Generate a podcast MP3 audio file research_name.mp3 from the research_name_podcast.md.
7. Send the MP3 and the PDF files.
